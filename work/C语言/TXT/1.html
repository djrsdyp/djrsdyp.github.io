数学函数

  1. abs(): 求整数的绝对值

int abs(int x);
/*
x为传入的参数，为int类型，可以为任意整数。如果x的值为正数，则返回x的值，
如果x的值为负数，则返回-x的值。
例如:abs(-5)返回5，abs(10)返回10
/*
  1. fabs(): 求浮点数的绝对值

double fabs(double x);
/*
x为传入的参数，为double类型，可以为正数、负数、零。
例如：fabs(-2.500000)返回2.500000，fabs(3.700000)返回3.700000
*/
  1. sqrt(): 求平方根

double sqrt(double x);
/*
参数x是需要求平方根的数字，函数返回x的平方根
例如：sqrt(16.0)返回4.000000，sqrt(25)返回5
*/
  1. pow(): 求次方

double pow(double x,double y);
/*
x是底数，y是指数。计算x的y次幂，并返回结果。
例子：pow(2.0,3.0)返回8.000000，pow(2.5,1.5)返回2.25
*/
  1. ceil(): 向上取整

double ceil(double x);
/*
参数x表示进行向上取整操作的实数。函数返回类型为double,即返回一个双精度浮点型结果。
例子：ceil(2.4)返回3.0，ceil(3.7)返回4.0
*/
  1. floor(): 向下取整

double floor(double x);
/*
例子：floor(3.6)返回3.0，floor(-3.6)返回-4.0
*/
  1. round(): 四舍五入

double round(double x);
/*
x是要四舍五入的浮点数，函数返回值为最接近x的整数。
例子：round(3.49)返回3.000000，round(-3.49)返回-3.000000
*/

补码、反码、原码

如果一个字节的第 8 位为 0，则表示该数是正数；如果第 8 位为 1，则表示该数是负数 ，一共8位。

举例：

5的原码为：00000101 -5的原码为：10000101

5的补码为：00000101 -5的补码为：11111011（最后加1）

5的反码为：00000101 -5的反码为：11111010
